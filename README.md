# 🤖 AIFindr – AI-Powered People Discovery Engine

AIFindr is a modern web application that uses AI to match people based on interests, goals, and personality traits. 🧠💫 Users can input natural language queries to find ideal matches from our database of profiles.

---

## ✨ Features

- 🔍 Natural language search to find people effortlessly  
- 🤖 AI-powered matching engine for accurate results  
- 📊 Score-based filtering for refined discovery  
- 🧬 Detailed match explanations with personality insights  
- ⚡ Real-time results as you type  
- 📱 Responsive design for all devices  
- 🎨 Beautiful UI with smooth animations  

---

## 🛠 Tech Stack

- ⚛️ React 18 with TypeScript  
- ⚡ Vite for blazing-fast development  
- 💨 Tailwind CSS for styling  
- 🧭 React Router for navigation  
- 🧩 Lucide React for icons

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
